=============================================
Coding Standards
=============================================

The following are the current iteration of Bashwork's Development Coding Standards policies.
All the documents are currently housed on the Application Architects' Sharepoint site 
`here <http://itsites/AppArch/default.aspx?RootFolder=%2fAppArch%2fShared%20Documents%2fStandards&FolderCTID=&View=%7b5A9753CF%2d91E6%2d4AA8%2d89D7%2d687FA06DA174%7d>`_.

Or, alternatively, you can click on any of the individual links below.  At the moment these
are simply links to the existing documents, though in the future we will attempt to convert
them to ReST so that they will be searchable through this site.
